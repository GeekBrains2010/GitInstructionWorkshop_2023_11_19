## Работа с удаленными репозиториями.
Для работы и изменений локально в удаленном репозиториии , обязательно нужно "подружить" VSC и GitHub, для этого нужно ввести следующие команды в терминале:
```sh
git config --global user.name "Your Name"
git config --global user.email mail@example.com
```
