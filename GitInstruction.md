# Инструкция по работе с git
Создание репозитория (начало отслеживания)
```sh
git init
```
Просмотр статуса
```sh
git status
```
Добавлене файла для последующего коммита 
```sh
git add <filename>
```
Коммит (закрепление изменений)
```sh
git commit -m "message text"
или
git commit -a {для принятия всех изменений} 
```
